# Como Contribuir

Ficamos felizes com seu interesse em contribuir! Por favor, siga estas diretrizes:
1. Abra uma 'Issue' para discutir a mudança que você deseja fazer.
2. Crie um 'Fork' do repositório e faça suas alterações em um branch separado.
3. Envie um 'Pull Request' com uma descrição clara das suas contribuições.
