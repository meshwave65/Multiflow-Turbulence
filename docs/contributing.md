# How to Contribute

We are excited about your interest in contributing! To ensure a smooth process, please follow these guidelines.

1.  **Open an 'Issue'**: Before starting work, please open an issue to discuss the change you wish to make. This allows for discussion and avoids duplicate effort.
2.  **Fork the Repository**: Create a fork of the project and make your changes in a separate branch.
3.  **Submit a 'Pull Request'**: Send a Pull Request with a clear description of your contributions.

# Como Contribuir

Ficamos felizes com seu interesse em contribuir! Para garantir um processo organizado, por favor, siga estas diretrizes:

1.  **Abra uma 'Issue'**: Antes de começar a trabalhar, por favor, abra uma issue para discutir a mudança que você deseja fazer. Isso permite a discussão e evita esforços duplicados.
2.  **Crie um 'Fork' do Repositório**: Crie um fork do projeto e faça suas alterações em um branch separado.
3.  **Envie um 'Pull Request'**: Envie um Pull Request com uma descrição clara das suas contribuições.
